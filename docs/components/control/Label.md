# Label

Location: `qml/components/control/display/Label.qml`

Theme text wrapper that provides Figma-mapped typography styles.

## Style API
- `style: title`
- `style: title2`
- `style: header`
- `style: header2`
- `style: body`
- `style: description`
- `style: caption`
- `style: disabled`

## Usage
```qml
UIF.Label { text: "Label"; style: body }
```
